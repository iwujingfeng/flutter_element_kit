# flutter_element_kit

### 便捷、高效、易用的Flutter组件库

### 更新日志
该组件库还处于开发阶段，预计今年中旬会正式开源。目前，还无法通过[pub](https://flutter.dev/developing-packages/)进行安装使用。按照我自己的计划，会将该组件库覆盖尽可能多的基础组件和业务组件。我会对常见布局和功能进行调查分析，封装出具有普遍适用性的组件。其目的就是让开发者可以更加高效地进行Flutter开发，享受优雅的编程体验。
